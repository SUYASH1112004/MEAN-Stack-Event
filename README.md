# MEAN-Stack-Event
Created a MEAN stack website for event registration where students registers for their events and login for registration of special events.
Any students from college can register for the event by their name and college id.
Students can also see for which events they have been registered by using their college id.

Students can also register for the special events by login is required for special event registration in which i have used jwt token.

So overall my project takes the data of registration from student to a specific event from client through client side framework angular and the by using node express it sends to backend and stores in database (Mongodb).
